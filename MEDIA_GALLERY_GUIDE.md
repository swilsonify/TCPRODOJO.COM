## ‚ûï How to Add Media

### **Step 1: Upload Your Photo/Video**

**For Photos:**
1. Go to https://imgur.com
2. Click "New post"
3. Upload your photo
4. Right-click image ‚Üí "Copy image address"

**For Videos:**
1. Upload to YouTube
2. Copy the video URL

### **Step 2: Add to Gallery Manager**

1. Click **"Add Media"** button
2. Fill in the form:
   - **Title**: Descriptive name (e.g., "Championship Night Hero")
   - **Section**: Choose where it appears (e.g., "Home - Hero Section")
   - **Type**: Image or Video
   - **Display Order**: Lower numbers appear first (0, 1, 2...)
   - **URL**: Paste your Imgur or YouTube URL
   - **Description**: Optional caption

3. Click **"Add Media"**

‚úÖ Done! The media is now available for that section.

## üîÑ How It Works with Your Website

### **Automatic Display**
Once you add media to a section, your website pages will automatically:
1. Fetch media for their section
2. Display items in order (by Display Order)
3. Show images as backgrounds, galleries, or inline photos
4. Embed videos as players

### **Multiple Items per Section**
You can add multiple photos/videos to any section:
- **Display Order 0**: Shows first
- **Display Order 1**: Shows second
- **Display Order 2**: Shows third
- etc.

## üí° Best Practices

### **For Hero Images (Banners)**
- Use landscape orientation (16:9 ratio)
- Recommended size: 1920x1080 pixels
- Keep file size under 2MB
- Use high-quality images for first impression

### **For Galleries**
- Use consistent sizing
- Add descriptive titles
- Use Display Order to arrange photos logically

### **For Videos**
- Upload to YouTube first (unlimited storage)
- Use short, engaging clips (under 2 minutes)
- Add clear titles and descriptions

### **Organization Tips**
- Use clear, descriptive titles
- Add descriptions for context
- Keep Display Order organized (0, 1, 2...)
- Delete unused media to keep gallery clean

## üÜò Troubleshooting

**Media not showing on website?**
- Check if URL is correct and accessible
- Verify you selected the right section
- Make sure Display Order is set
- Try redeploying the website

**Image broken/not loading?**
- Test URL in browser directly
- Re-upload to Imgur and get new URL
- Check file size (keep under 5MB)

**Video not playing?**
- Make sure video is set to "Public" on YouTube
- Use full YouTube URL (not shortened)
- Test URL in browser
